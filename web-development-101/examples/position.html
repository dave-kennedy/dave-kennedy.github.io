Position

By default, all elements have a position property set to static.

Setting an element's position property to absolute allows us to place it exactly where we want on the page. The top, right, bottom, and left properties specify where the corresponding edge should be placed relative to the page.

The position property of this element isn't specified, so it's static by default.
The position property of this element is absolute. Its top-left corner is 300 pixels from the top-left corner of the page.

Setting an element's position property to relative allows us to move it up, down, left, or right. The top, right, bottom, and left properties specify where the corresponding edge should be placed relative to its default position.

The position property of this element isn't specified, so it's static by default.
The position property of this element is relative. Its top-left corner is shifted 40 pixels down and right.

Setting an element's position property to absolute when its parent's position property is relative allows us to place the element exactly where we want within its parent. In this case, the top, right, bottom, and left properties specify where the corresponding edge should be placed relative to its parent.
