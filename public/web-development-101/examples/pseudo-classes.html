Pseudo-classes

We can use psuedo-classes to style an element based on its state.

:hover

This pseudo-class applies to an element when the user's mouse cursor is moved over it. Don't depend on this state for important functionality. Not all devices have a mouse, e.g. phones and screen readers.

Link

:active

This pseudo-class applies to an element when the user clicks or taps on it. When the user lifts their finger off the mouse button or touch screen, the element is no longer active.

Link

:focus

This pseudo-class applies to an element when the user clicks or taps on it or uses the tab key to select it. The element retains focus until the user clicks or taps elsewhere or hits the tab key again.
