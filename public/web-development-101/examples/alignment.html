Alignment

text-align

This property only applies to a container's content. It has no effect on void elements. To center a void element, wrap it in a div element, then change the parent's text-align property to center.

float

This property pulls an element out of its default position to the left or right, allowing other content to "wrap around" it until it's "cleared" by another element with a clear property set to left, right, or both.
