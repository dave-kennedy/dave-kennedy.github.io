Spacing

border

An element's border is composed of three properties: border-width, border-style, and border-color. 

border-width: 5px;
border-style: solid;
border-color: red;

All three properties can be set at once by assigning multiple values to the border property:

border: 5px solid red;

margin

This property controls the amount of space around an element.

margin: 40px;

The space on each side can be controlled separately with the margin-top, margin-right, margin-bottom, and margin-left properties.

margin-top: 40px;
margin-right: 40px;
margin-bottom: 40px;
margin-left: 40px;

auto

Setting a block-level element's margin-left and margin-right properties to auto centers the element horizontally within its parent. 

display: block;
margin-left: auto;
margin-right: auto;

padding

This property controls the amount of space inside an element.

padding: 40px;

The space on each side can be controlled separately with the padding-top, padding-right, padding-bottom, and padding-left properties.

padding-top: 40px;
padding-right: 40px;
padding-bottom: 40px;
padding-left: 40px;
